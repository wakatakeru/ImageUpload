# ImageUpload
## このアプリケーションについて
- 小規模グループの画像共有に利用できます
- イラストなどを共有するアプリケーションです
- サムネイルによる画像一覧表示が利用できます
- アップロードしたユーザを参照できます
- **現在開発中です**

## インストールと起動
1. `$ bundle install --path=vendor/bundle`
2. `$ bundle exec rake db:migrate`
3. `$ bundle exec rails s`

## 今後の実装機能について
- コメント機能とfav機能
- タイトル検索機能
- 管理者によるアカウント管理機能
- ユーザ情報の編集機能
- 投稿の編集機能
